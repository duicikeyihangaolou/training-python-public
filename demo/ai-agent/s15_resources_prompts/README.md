# s15: Resources 与 Prompts

> 对应讲义：[we-know-ai-agent.md](../../doc/we-know-ai-agent.md#15-resources-与-prompts选讲)

本节介绍 MCP 的 Resources 和 Prompts 能力。

## 核心概念

**Resources**：只读数据（文件/配置/API 响应），通过 `resources/list` / `resources/read` 暴露

**Prompts**：预定义提示模板，供 Client 获取后填变量

## 文件说明

| 文件                  | 说明                     |
| ------------------- | ---------------------- |
| `resources_demo.py` | Resources 与 Prompts 示例 |

## 运行方式

```bash
cd s15_resources_prompts
python resources_demo.py
```
